Guía rápida · Claude + GitHub

Agrega @claude a
cualquier repo

3 pasos para tener a Claude respondiendo en tus issues y pull requests.

01
Instala la GitHub App de Claude Solo 1 vez
Solo se hace una vez. Puedes instalarla en todos tus repos o solo en uno específico.
Ir a github.com/apps/claude → Click "Install"
Selecciona tu cuenta u organización
Elige "All repositories" o selecciona el repo específico
Click "Install & Authorize"
02
Crea tu API Key de Anthropic Solo 1 vez
Necesitas una key para que Claude pueda procesar las solicitudes del repo.
console.anthropic.com → Settings → API Keys
Click "Create Key" → ponle un nombre (ej: github-mi-repo)
Copia la key — empieza con sk-ant-...
⚠️ Solo se muestra una vez — guárdala antes de cerrar
03
Agrega el Secret al repo Por repo
Cada repo donde quieras usar @claude necesita este secret.
Settings → Secrets and variables → Actions
Click "New repository secret"
Name: ANTHROPIC_API_KEY (exactamente así)
Secret: pega tu key sk-ant-...
Click "Add secret"
04
Crea el archivo de workflow Por repo
Crea este archivo en tu repo. Puedes hacerlo directo desde GitHub sin tocar la terminal.
En tu repo, crea el archivo .github/workflows/claude.yml
Pega el siguiente contenido exacto:
copiar name: Claude Code on: issue_comment: types: [created] pull_request_review_comment: types: [created] jobs: claude: runs-on: ubuntu-latest permissions: contents: write issues: write pull-requests: write id-token: write steps: - uses: actions/checkout@v4 with: fetch-depth: 1 - uses: anthropics/claude-code-action@v1 with: anthropic_api_key: ${{ secrets.ANTHROPIC_API_KEY }}
Haz commit del archivo → Click "Commit changes"
05
¡Listo! Prueba mencionando @claude
Ve a cualquier Issue o Pull Request del repo y escribe un comentario.
Abre un Issue o Pull Request en tu repo
Escribe un comentario mencionando @claude
# Ejemplos de uso: @claude explícame qué hace este archivo @claude review this PR @claude fix the bug in line 42 @claude sugiere mejoras para este código
✅
Para agregar Claude a un nuevo repo solo repites los pasos 3 y 4
La GitHub App (paso 1) y la API Key (paso 2) son de una sola vez. Para cada repo nuevo solo necesitas agregar el secret ANTHROPIC_API_KEY y copiar el archivo claude.yml.
